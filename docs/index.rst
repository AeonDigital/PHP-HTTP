Instalação
----------

Instale em seu projeto usando o composer:


**Via terminal**

.. code-block:: console

    composer require aeondigital/phphttp


**Via composer.json**

.. code-block:: json

    "require": {
        "aeondigital/phphttp": "dev-master"
    }
